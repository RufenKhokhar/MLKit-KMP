# MLKit-KMP

Kotlin Multiplatform bindings for Google ML Kit. Shared Kotlin API over ML Kit’s native SDKs so you can integrate Vision and NLP once and run on Android and iOS. It handles platform interop, image inputs, threading, and results mapping, with Swift-friendly symbols on iOS and simple Gradle setup on Android.
