# MLKit-KMP

Kotlin Multiplatform bindings for Google ML Kit. This library wraps ML Kit's native SDKs behind a clean, shared Kotlin API, so you can integrate vision features once and run them on Android and iOS. It handles platform interop, image inputs, threading, and results mapping for you.

## Available in v0.2.0

| Feature | Artifact |
|---|---|
| Barcode Scanning | `mlkit-barcode` / `mlkit-barcode-play-services` |
| Face Detection | `mlkit-face-detection` / `mlkit-face-detection-play-services` |
| Text Recognition | `mlkit-text-recognition` / `mlkit-text-recognition-play-services` |
| Pose Detection | `mlkit-pose-detection` |
